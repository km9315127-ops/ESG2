淨零碳規劃管理師 刷題系統

淨零碳規劃管理師（iPAS）模擬測驗與練習平台，支援科目一與科目二刷題、每次至少 25 題、含複選題、解析與錯題追蹤。

科目一：節能減碳技術實務

涵蓋能源管理、節能減碳、低碳技術與實務應用。

科目二：碳管理制度實務

涵蓋碳盤查、碳管理制度、揭露、交易與治理應用。

單一 HTML + JSON 題庫版雛形架構

1. 前端唯一檔案：index.html - 負責畫面、出題流程、判分、解析、成績顯示 - 不依賴後端即可展示 2. 題庫檔：questions.json - 存所有正式題目 - 每題含 subject / chapter / topic / difficulty / type / answer / explanation 3. 本機資料暫存：localStorage - 保存錯題本 - 保存最近一次成績 - 保存練習歷程摘要 4. 未來升級接口 - 將 fetch('questions.json') 改為 fetch('/api/questions') - 將 localStorage 改為資料庫 - 將解析生成改為 LLM API

type 建議使用：single_choice、multiple_choice、scenario、trap。
